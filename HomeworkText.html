Text Formatting:

They are multiple ways to format text in HTML without the use of CSS

But I did it wilth CSS anyway :)

1. In Hyper Text Markup Language you can bold text by using <b>

Example:
This text has applied bold to it

Pro tip!
You can also make text strong with <strong>


2. In Hyper Text Markup Language you can make text italic by using <i>

Example:
This text has applied italics to it

Pro tip!
You can also emphasize text with <em>


3. In Hyper Text Markup Language you can make text subscripit by using <sub>

Example:
Did you know water in chemestry is H2O?


4. In Hyper Text Markup Language you can make text supscript by using <sup>

Example:
Did you know that every number to the power of 0 is 1? (3020 is 1)


5. In Hyper Text Markup Language you can mark text by using <mark>

Example:
Usually in school they make us mark text in our books

Pro tip!
You can change the color of <mark> using CSS!
