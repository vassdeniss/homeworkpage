Hyper Text Markup Language - HTML is used to make web pages for the world wide web

The components of making a fully working web page are three - HTML, CSS and JS (Hyper Text Markup Language, Cascading Style Sheets and JavaScript)

For now lets concentrate on HTML only. Writing HTML is very easy and ill show you how!

For the sake of ease I'll write HTML code in red 

 
            <!DOCTYPE html>
            <html>
            <head>
            <title>Example Title</title>
            </head>
            <body>
            <h1>Example Header</h1>
            <p>Example Text</p>
            </body>
            </html>

This is a working example of a HTML file structure. I'll now explain each of the things you see.

But first lemme fix this mess...

Much easier to read right? That's one way I recommend to format your code - by simply spacing and tabbing

Now lets explain it!

Whats a tag in HTML? A tag is written inside < >. You must end tags by typing </ >. So to write content you open a tag, write code, close tag!

<!DOCTYPE html> - Simply says to the browser that this document is an HTML document. Not required but recommended

<html> - There is not much to know about this tag. You must have it and all your code must be between it.

<head> - You can customize CSS inside the head, load external CSS, load page icons and much more!

<title> - The name of the web page that apears on top on the little tab in your browser!

<body> - The literal body of your page! Everything that your page will containg goes between <body>

<h1> - Stands for "header" and is the big bolded title on your page! You can use <h1> through <h6> to make more headers that are smaller in size!

<p> - Stands for "paragraph" and used to write text on your page! You can of course add a lot of styling to the paragraphs but leave that for now! If you want to add a break to the paragraph and start a new line use this:
Example:
<p> This is line one. <br> And now this will be line two! </p>
