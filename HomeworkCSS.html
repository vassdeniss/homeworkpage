Styling with CSS

Usually I use my own external css but not today I guess...

You can write CSS inside the HTML file using <style>

You can write external CSS using a .css file and typing
<link rel="stylesheet" href="location">
in the head (location - location of the css file)

You can id tags with id=""... And then change them in the CSS with #!

You can also make classes with class="" and then change them in the CSS with .!

And change multiple things...

At the same time with that class!
